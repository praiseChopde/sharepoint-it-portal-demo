# City IT Support Portal — SharePoint Online Demo

A fully functional IT help desk portal built in Microsoft SharePoint Online, modeled after a real municipal IT support environment. This project demonstrates hands-on experience with SharePoint site administration, document library management, list-based ticket tracking, and IT asset management.

---

## Why I Built This

The City of Saint John IT team supports hundreds of municipal staff across departments including Planning, Engineering, Finance, and Public Works. I built this portal to demonstrate practical familiarity with the Microsoft 365 / SharePoint platform used in enterprise and government IT environments — going beyond reading about it to actually configuring and populating a working system.


### Home Page
Custom-branded IT support portal for City of Saint John staff with:
- Quick-action buttons: Submit a ticket, View ticket status, Alerts and known issues
- Department-specific IT announcements
- Direct links to self-help resources and guides


### Tickets List — Help Desk Issue Tracker
A structured issue tracker with columns for Issue, Description, Priority, Status, and Assignment — mirroring a real service desk intake workflow.

Sample tickets include:
- Laptop Wi-Fi failure after Windows update (High / In Progress)
- Expired password preventing new employee login (High / Resolved)
- Outlook not receiving emails (Medium / In Progress)
- Hardware request: second monitor (Low / Open)
- Intune MDM enrollment failure on City iPhone (Medium / Open)

### Devices List — IT Asset Manager
A full asset tracking register with fields for Manufacturer, Model, Asset Type, Status, Serial Number, and Purchase Date.

Sample assets tracked:
- Dell Latitude 5540 laptops (In use / In repair)
- HP EliteDesk 800 G9 desktop
- Apple iPhone 14 (Mobile / In use)
- Microsoft Surface Pro 9 tablet (Available)



## Platform & Tools

- **Microsoft SharePoint Online** (Microsoft 365)
- **SharePoint IT Help Desk template** — configured and customized
- **SharePoint Lists** — Tickets and Devices (custom columns, status badges)
- **SharePoint Document Library** — IT Guides with versioning enabled
- **Microsoft Word Online** — guide documents authored directly in browser


## Key Concepts Demonstrated

- SharePoint site creation and administration
- Document library organization and management
- List-based data entry and structured ticket intake
- IT asset tracking with status lifecycle (Available → In use → In repair)
- Staff-facing knowledge base design
- Microsoft 365 ecosystem familiarity (SharePoint, Word Online, Intune MDM concepts)
- Real-world IT support workflow understanding: triage, escalation, resolution
